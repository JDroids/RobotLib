singlePoleIIR

fun singlePoleIIR(source: PIDSource, timeConstant: Double, period: Double): LinearDigitalFilter

Creates a one-pole IIR low-pass filter of the form: yn = (1-gain)xn+gainyn-1 where gain = e^(-dt / T), T is the time constant in seconds.

This filter is stable for time constants greater than zero.

Parameters

source - The PIDSource object that is used to get values

timeConstant - The discrete-time time constant in seconds

period - The period in seconds between samples taken by the user
