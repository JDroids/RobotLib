fun movingAverage(source: PIDSource, taps: Int): LinearDigitalFilter

Creates a K-tap FIR moving average filter of the form: yn = 1/k * (xk + xk-1 + ...

  • x0).

This filter is always stable.

Parameters

source - The PIDSource object that is used to get values

taps - The numbers of samples to average over. Higher = smoother but also slower.

Exceptions

IllegalArgumentException - if the number of taps is less than 1
