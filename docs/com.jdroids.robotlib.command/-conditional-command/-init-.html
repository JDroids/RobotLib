ConditionalCommand(onTrue: Command, onFalse: Command? = null)

Creates a new ConditionalCommand with given onTrue and onFalse Commands. onFalse can be left unspecified if you do not want it to be a no-op on the false condition.

Parameters

onTrue - the command to run when the condition function returns true

onFalse - the command to run when the condition function returns true
