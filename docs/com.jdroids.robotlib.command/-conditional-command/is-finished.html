isFinished

protected open fun isFinished(): Boolean

Overrides Command.isFinished

Returns whether this command is finished. If it is, then the command will be removed and end will be called.

It may be useful to reference isTimedOut for time-sensitive commands.

Returning false will result in the command never ending automatically. It may still be canceled manually or interrupted by another command. Returning true will result in the command executing once and finishing immediately. It is recommended to use InstantCommand for that.
