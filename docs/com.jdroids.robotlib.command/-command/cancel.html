cancel

fun cancel(): Unit

This will cancel the current command.

This will cancel the current command eventually. It can be called multiple times. And it can be called when the command is not running. If the command is running however, then the command will be marked as canceled and eventually removed.

A command can not be canceled if it is a part of a command group, you must cancel te command group instead.

Exceptions

IllegalUseOfCommandException - if this command is a part of a command group
