requires

protected fun requires(subsystem: Subsystem): Unit

This method specifies that the given Subsystem is used by this command. This method is crucial to the functioning of the Command System in general.

Note that the recommended way to call this method is in the constructor.

Parameters

subsystem - the required Subsystem

Exceptions

IllegalUseOfCommandException - if this command has been started before or if it has been given to a command group
