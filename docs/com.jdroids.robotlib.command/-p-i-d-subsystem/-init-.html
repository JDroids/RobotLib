PIDSubsystem(input: () -> Double, output: (Double) -> Unit, pidCoefficients: <ERROR CLASS>, v: Double = 0.0, a: Double = 0.0)

This class is designed to handle the case where there is a Subsystem which uses a single SimplePIDController almost constantly (for instance, an elevator which attempts to stay at a constant height).

It provides some convinience methods to run an internal SimplePIDController. It also allows access to the internal SimplePIDController in order to give total control to the programmer.
