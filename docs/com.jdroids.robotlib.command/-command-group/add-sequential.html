fun addSequential(command: Command): Unit

Adds a new Command to the group. The Command will be started after all the previously added commands.

Note that any requirement the given Command has will be added to the group. For this reason, a command's requirements can not be changed after being added to a group.

It is recommended that this method be called in the constructor.

fun addSequential(command: Command, timeout: Double): Unit

Adds a new Command to the group with a given timeout. The Command will be started after all the previously added commands.

Once the Command is started, it will run until it finishes or the time expires, whichever is sooner. Note that the given Command will have no knowledge that it is on a timer.

Note that any requirement the given Command has will be added to the group. For this reason, a command's requirements can not be changed after being added to a group.

It is recommended that this method be called in the constructor.

Parameters

command - the command to be added

timeout - The timeout (in seconds)

Exceptions

IllegalUseOfCommandException - if the group has been started before or been given to another group

IllegalArgumentException - if the timeout is less than 0
