Subsystem()

This class defines a major component of the robot.

A good example of a subsystem is the drivetrain, or a claw if the robot has one.

All actuators should be part of a subsystem. For instance, all the motors powering wheels should be a part of some sort of a "Drivetrain" subsystem.

Subsystems are used within the command system as requirements for commands. Only one command which requires a subsystem can run at a time. Also, subsystems can have default commands which are started if there is no command running which requires this subsystem.
