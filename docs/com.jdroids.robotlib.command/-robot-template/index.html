RobotTemplate

abstract class RobotTemplate : Any

RobotTemplate is a class designed to be a base for team's Robot classes for use as part of the Command system. Each robot should instantiate each subsystem. The Robot class is also a candidate for being made as a singleton, as you only have one robot.
