setContinuous

fun setContinuous(continuous: Boolean = true): Unit

Set the PID controller to consider the input to be continuous. Rather than using the max and min input as constraints, it considers them to be the same point and automatically calculates the shortest route to the setpoint.

Parameters

continuous - Set to true it turns on continuous, set to false it turns off continuous (defaults to true)
