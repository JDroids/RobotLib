pidWrite

open fun pidWrite(output: Double): Unit

Overrides PIDOutput.pidWrite

Passes the output directly to setSetpoint().

PIDControllers can be nested by passing a SimplePIDController as another SimplePIDController's output. In that case, the output of the parent controller becomes the input (i.e., the reference) of the child.

It is the caller's responsibility to put the data into a valid form for setSetpoint().
