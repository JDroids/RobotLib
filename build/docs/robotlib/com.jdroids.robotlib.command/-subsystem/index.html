Subsystem

interface Subsystem

The Subsystem interface represents a physical component of a robot. These can be a drivetrain, a claw, etc. They should be registered at some point.

Functions

initHardware

 abstract fun initHardware(): Unit

This method should initialize the hardware of the component. This is typically getting all the hardware devices from hardware map.

periodic

 abstract fun periodic(): Unit

This method is called by Scheduler.periodic as long as the subsystem is registered.
